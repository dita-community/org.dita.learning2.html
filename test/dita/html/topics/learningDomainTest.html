1 Learning Domain Test

Provides test cases for learning domain interaction elements (questions and answers)

The "learning 1" domain provides interactions (assessments or questions and answers) as defined in DITA 1.2. The "learning 2" domain is new with DITA 1.3 and extends the content models of the assessments to allow block elements in questions, answers, and feedback.
